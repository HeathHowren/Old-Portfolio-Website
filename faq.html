Learn Game Hacking & Coding

Q: How do I make a wallhack?
A: Any “ESP” tutorial will do, you will also need to have a basic understanding of coding knowledge first. Most games operate similarly in the way coordinates work so you can transfer knowledge to tothe games. The most similar games are ones built with the same engine (ex: Unity and Unreal Engine).

Q: How do I compile a cheat in VS?
A: To do this you will need to go to the “build” tab in VS and then select your parameters as “release” and CPU as “x64” or “x86” depending on the architecture of the application you are planning to read/write memory to.
Q: How long until I can make a cheat for a game like Rainbow Six Siege?
A: If you want to actually learn game hacking and move onto games with anti-cheats like BattlEye and Easy Anti-Cheat then most likely you will need several years of game hacking knowledge, this can be accomplished by first fully learning your language of choice then moving onto the game hacking part.
Q: Which programming language should I learn C# or C++?
A: It is mostly up to preference, but I would say C++ is a good first language for beginners when it comes to low-level programming.
Q: What is a bypass? How is it different from just reading and writing memory?
A: There are multiple types of bypasses. The goal of a bypass is to allow the cheat application to read/write to the target application undetected.
Q: Is C# game hacking possible?
A: Yes it is, although most people use C# for the pretty interfaces and it’s “amazing syntax” it usually isn’t a cheater’s first choice for game hacking. Although there aren’t a lot of tutorials and isn’t a lot of information out there on C# game hacking is definitely a viable option from games like CS:GO to Rainbow Six Siege.
Q: Internal Hacks vs. External Hacks?
A: It truly depends on the game that you’re going to hack. If you are looking at CS:GO definitely start out with Internals as there are a lot more resources on CS:GO internals compared to externals. For games with beefier anti-cheats like BE and EAC you would most probably want to start with externals as they might be easier to grasp, but it really depends on the Language and the Game you want to cheat in.
